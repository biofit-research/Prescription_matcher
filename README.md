# ℞ 처방약 매칭 도우미

이 프로젝트는 타 병원의 처방약을 본원에서 보유 중인 약품과 성분·함량별로 매칭해주는 도구입니다.[cite: 1]

## 주요 기능
* **자동 매칭**: 상품명, 성분명, 함량을 입력하면 본원 약품 마스터와 대조합니다.[cite: 1]
* **대체약 검색**: 동일 상품이 없을 경우 성분과 함량이 일치하는 대체 의약품을 찾아줍니다.[cite: 1]
* **OCR 입력**: 처방전 사진이나 메모를 촬영하여 텍스트를 자동으로 입력할 수 있습니다.[cite: 1]
* **재고 상태 확인**: 입원·외래 가능 여부 및 품절 상태를 표시합니다.[cite: 1]

## 사용 방법
1. 한 줄에 한 약품씩 입력합니다. (예: `노바스크정 5mg | amlodipine | 5mg`)[cite: 1]
2. '매칭 시작' 버튼을 눌러 결과를 확인합니다.[cite: 1]
